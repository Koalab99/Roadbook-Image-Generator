Known bugs

On firefox, you might experience issues with image copying.

To fix it, you need to :

  1. Open a new tab (using Ctrl + T)
  2. Type about:config
  3. If requested, accept the risks and continue
  4. Enable dom.events.asyncClipboard.clipboardItem
  5. Enable dom.events.testing.asyncClipboard
